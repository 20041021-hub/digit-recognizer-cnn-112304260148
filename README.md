## 6. 实验方法说明

### （1）数据预处理
1. 数据加载：使用pandas读取CSV格式的训练集和测试集数据
2. 数据归一化：将像素值从0-255归一化到0-1范围，提高模型训练效率
3. 数据划分：训练集用于模型训练，测试集用于生成预测结果

### （2）模型架构
使用简单的深度神经网络（DNN），网络结构如下：
- 输入层：784个神经元（对应28×28像素的灰度图片）
- 隐藏层1：128个神经元，使用ReLU激活函数
- 隐藏层2：64个神经元，使用ReLU激活函数
- 输出层：10个神经元（对应0-9共10个数字）

### （3）模型训练
- 优化器：Adam，学习率0.001
- 损失函数：交叉熵损失
- 批量大小：64
- 训练轮数：10

## 9. 实验总结
**实验过程：**
1. 首先加载并预处理数据，将像素值归一化到0-1范围
2. 构建了一个包含两个隐藏层的深度神经网络模型
3. 使用Adam优化器和交叉熵损失函数训练模型
4. 训练过程中，模型准确率逐步提高，最终达到99.15%
5. 在测试集上生成预测结果，并生成符合Kaggle要求的提交文件

**遇到的问题及解决方法：**
1. PyTorch版本问题：初始安装的是CPU版本，后来根据显卡型号安装了GPU版本，提高了训练速度
2. 内存管理：处理大量数据时，通过合理设置批量大小避免内存溢出
3. 模型选择：尝试了不同的网络结构，最终选择了两层隐藏层的结构，在准确率和训练时间之间取得了平衡

**实验结果分析：**
- 模型在训练集上的最终准确率达到99.15%，说明模型拟合效果良好
- 训练过程中损失函数值持续下降，从0.4098下降到0.0257，表明模型学习效果明显
- 模型结构简单，训练时间短，适合快速部署和应用

**改进方向：**
1. 尝试使用更复杂的模型结构，如CNN（卷积神经网络），进一步提高识别准确率
2. 增加数据增强步骤，如旋转、平移等，提高模型的泛化能力
3. 调整超参数，如学习率、批量大小、训练轮数等，进一步优化模型性能
4. 尝试集成学习，结合多个模型的预测结果，提高最终预测的准确性
